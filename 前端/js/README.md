# bind和call的区别:
bind是在执行时将一个对象放在一个对象上执行，但并没有立即执行，其返回值是一个绑定后的函数，而call则是在执行时将一个对象放在一个对象上执行，并立即执行。其返回值为函数执行后的返回值。
