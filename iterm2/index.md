# 常用配置
1.以热键快速打开/隐藏
2.使用前一标签的当前路径作为新tab的当前路径:去profile设置
3.自定义配色方案：Preferences->Profiles->Colors

# 常用功能
1.标签
(1)新建: command + t
(2)关闭: command + 2
(3)切换: command + 左右方向键/数字

2.分屏
(1)新建: command + d / command + shift + d
(2)关闭: 
(3)切换: command + [ / ]

3.全屏：command + enter
4.查找：command + f
5.清屏：command + r
6.选择即复制

# 高级功能
1.查看历史命令记录：command + ; //自动完成
2.查看剪贴板历史记录：command + shift + h //自动完成
3.搜索命令历史：ctrl + r
4.到行首：ctrl + a
5.到行尾：ctrl + e
6.ctrl + f/b
