# IP协议的功能
1.寻址和路由：根据对方的IP地址，寻找最佳路径传输路径。
2.传递服务：① 不可靠（IP协议只是尽自己最大努力去传输数据包），可靠性由上层协议提供（TCP协议）；② 无连接；（事先不建立会话）
3.拥塞控制：防止数据包传输的过程当中出现拥塞现象，造成数据包的丢失和延迟

# Internet地址结构

## 表示IP地址
